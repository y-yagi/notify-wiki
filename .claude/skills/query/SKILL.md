`$ARGUMENTS` の質問に、`wiki/` に書かれている内容のみを根拠に答える。

## 手順

1. `wiki/index.md` で全体像を把握し、関連しそうな `wiki/concepts/*.md`・`wiki/comparisons/*.md` を探して読む。
2. 見つかった内容に基づいて質問に回答する。回答の根拠になったページを `[[concept-name]]` の形式か、ファイルパスで明示する。
3. 複数ページにまたがる場合は、矛盾がないか確認してから統合して答える。矛盾を見つけた場合はそれ自体を報告する（推測で解消しない）。
4. wiki内に該当する記述が無い、または不十分な場合は、素直に「wiki にはまだこの情報がありません」と伝える。決して知識で埋めて断定しない。
   - その上で、どんな一次資料（man page、RFC、ブログ記事など）を `raw/` に追加すれば解決できそうかを提案する。
5. `wiki/` の内容を裏付ける一般知識（学習済み知識）で補足したい場合は、wikiの記述とは明確に区別して「※wiki外の一般知識」等と明記する。

## 制約

- このskillはread-only。`wiki/` や `raw/` のファイルを編集しない。編集が必要な発見（誤記・矛盾等）があれば、回答の最後に触れるだけに留め、実際の修正はユーザーの指示を待つか `/lint` に委ねる。
